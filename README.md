# obsidian-knowledge-base

公開を前提にした Obsidian の知識ベースです。  
日々のメモ、調査記録、記事の草案を蓄積し、必要に応じて整理・公開するためのリポジトリとして運用します。

このリポジトリでは、最初から細かく分類しすぎず、まずは日付つきの個別ノートとして書き溜め、後から必要に応じて再整理する方針を採用しています。  
そのため、フォルダー構成は最小限に保ち、検索・リンク・タグ・再編集を重視します。

## Purpose

このリポジトリの目的は次のとおりです。

- 日々の学習メモや調査記録を蓄積する
- 記事や公開ドキュメントの草案を育てる
- 後から再利用しやすい形で知識を整理する
- Obsidian を使った公開前提の知識管理を行う
