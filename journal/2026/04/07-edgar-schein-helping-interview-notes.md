# 概要

エドガー・H・シャインの『人を助けるとはどういうことか――本当の「協力関係」をつくる7つの原則』を、転職面談で活用するためのメモ。

『チームが機能するとはどういうことか』や『問いかける技術』を読んだ後に本書を読み返すと、著者がチームビルディングをどのように理解しているかが見えやすい。特に印象に残ったのは、チームビルディングの目的を「メンバー全員における人間関係を作ること」と捉えている点である。

本書では、チームのメンバーが次の4つの問題について安心感を得られる状況をつくることが重要だとされる。

# 4つの観点

## 1. 自分の役割は何か

「私はどんな人間になればいいのか。このグループでの私の役割は何か。」

これは職場やチームでいえば、期待役割の明確さに関わる。自分が何を期待されているのかが曖昧なままだと、能力の問題というより、不安や遠慮によって動きづらくなる。

面談では、次のように言い換えられる。

- 良いチームでは、各メンバーが何を期待されているかを理解しやすい
- 役割が明確だと、自分の貢献の仕方を考えやすい
- チームに入るときは、まず期待役割の理解を重視する

## 2. 自分はどれくらい影響を及ぼせるか

「このグループで、私はどれくらいのコントロール、あるいは影響を及ぼすことになるか。」

これは発言権や裁量、意思決定への参加可能性に関わる。何をどこまで決めてよいのか、どこで相談すべきかがわからないと、人は過剰に萎縮するか、逆に不用意に踏み込みすぎる。

面談では、次のように整理できる。

- 良いチームでは、判断の境界や相談のラインが見えやすい
- 自分の意見がどのように扱われるかがわかることが安心につながる
- 協力関係には、心理的安全性だけでなく、影響範囲の見通しも必要である

## 3. 自分の目標や要求を果たせるか

「このグループで、私は自分の目標、あるいは要求を果たすことができるか。」

これは個人の関心や成長と、チームの目標の接続に関わる。チームへの所属が、個人の目標の放棄を意味してしまうと、表面上は協力していても持続しにくい。

面談では、わがままではなく、次のような観点として話せる。

- 自分の関心や強みが、チームへの貢献とどうつながるかを大切にしたい
- 個人の成長とチームの成果が対立しすぎない環境が望ましい
- 協力関係は、個人が何を大切にしているかを無視しないことから始まる

## 4. どれくらい親しくなれるか

「このグループで、人々はどれくらい親しくなるだろうか。」

これは単なる仲良しの話ではなく、困ったときに助けを求められるか、率直に相談できるかという関係の近さに関わる。協力関係は、形式的な役割分担だけでは成り立たず、一定の信頼が必要になる。

面談では、次のように表現できる。

- 良いチームでは、問題を早めに相談しやすい
- 間違いや不明点を隠さずに共有できることが重要である
- 親しさは目的ではなく、協力を成立させる基盤である

# 面談で使える要点

## チームワーク観の整理

この本を踏まえると、チームワークは「仲良くすること」や「空気を読むこと」ではなく、安心して協力できる条件を整えることだと整理できる。

自分の理解では、良いチームには次の4点が必要である。

- 役割が見える
- 影響範囲が見える
- 個人の目標とチームの目標がつながる
- 困ったときに助けを求められる関係がある

## 面談での言い方の例

### どんなチームが良いチームだと思うか

良いチームとは、単に優秀な人が集まっている状態ではなく、メンバーが安心して協力できる状態だと考えています。具体的には、自分の役割がわかること、どこまで影響を及ぼせるかが見えること、自分の目標とチームの目標が接続していること、そして困ったときに相談しやすい関係があることです。

### チームで大事にしていることは何か

私は、最初から強く主張して場を動かすことよりも、まず期待役割や判断の境界を確認し、相談しやすい関係をつくることを重視しています。そのほうが結果的に協力関係が安定しやすいと考えています。

### 意見が違う人とどう関わるか

意見の違いそのものよりも、違いを安全に扱える関係があるかどうかを重視しています。役割や目的、判断権限が曖昧だと対立が感情的になりやすいので、まず前提をそろえ、必要に応じて相手の意図や背景を確認するようにしています。

# 自分に引きつけた学び

自分はこれまで反感を買ったり、対立を生みやすかったと感じることがあるが、それは単に伝え方の問題だけではなく、相手が安心して協力できる条件を十分につくれていなかった可能性がある。

この本の観点を使うと、チームや対話の問題を人格論ではなく、関係設計の問題として見直しやすい。

特に重要なのは次の点である。

- 相手にとって自分の役割がどう見えているか
- 相手がどこまで発言してよいと感じているか
- 相手自身の目標や事情を無視していないか
- 助けを求めても大丈夫だと思える関係になっているか

# 面談準備への応用

転職面談では、自分の強みを個人能力だけで語るのではなく、協力関係をどう見ているかという観点でも語れるようにしたい。

その際は、次の流れで話すと整理しやすい。

1. 良いチームの条件について自分なりの考えを述べる
2. その条件を、自分はどう支えようとしているかを述べる
3. 過去の経験で、それが必要だと感じた場面を簡潔に添える

# 一言でまとめるなら

チームワークとは、仲の良さの演出ではなく、メンバーが役割・影響・目標・関係性について安心感を持てるようにすることである。
